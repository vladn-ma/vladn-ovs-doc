33  * |1|-|-|-|1|-|-|-|R|D|R|R|A|R|R|R| Group Policy ID |
34  * +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
35  * | VXLAN Network Identifier (VNI) | Reserved |
36  * +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
37  *
38  * D = Don't Learn bit. When set, this bit indicates that the egress
